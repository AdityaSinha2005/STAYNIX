# StayNix

StayNix is a full-stack MERN based accommodation listing platform inspired by travel applications. Users can explore places, create their own listings, upload images, view locations on maps, and add reviews.

---

## Main Features

- User authentication system (Sign Up, Login, Logout)
- Create, edit and delete listings
- Image upload using Cloudinary
- Search and category filters
- Review and rating system
- Interactive map integration
- Responsive design for mobile and desktop

---

## Tech Stack

Frontend:
- HTML
- CSS
- Bootstrap
- EJS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas
- Mongoose

Authentication:
- Passport.js
- Express Session

Image Handling:
- Multer
- Cloudinary
- Multer Storage Cloudinary

Maps:
- Leaflet.js
- OpenStreetMap

## Installation

Clone repository:

```bash
git clone https://github.com/AdityaSinha2005/STAYNIX.git
```

Move into folder:

```bash
cd STAYNIX
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
node app.js
```

or

```bash
nodemon app.js
```
